# MVP Grupo-3

# Simulador de Impacto Económico Global

## Introducción

Este proyecto nace con el objetivo de desarrollar una herramienta de simulación económica simple, visual e interactiva que permita comprender cómo distintos eventos globales pueden impactar sobre variables económicas relevantes.

Nuestro MVP (Minimum Viable Product) busca mantener el concepto principal de la idea original, pero con un alcance más acotado y realista para su desarrollo inicial.

La plataforma permitirá al usuario seleccionar eventos predefinidos —como crisis económicas, conflictos internacionales o cambios políticos— y visualizar cómo estos afectan indicadores económicos globales mediante simulaciones simples y comprensibles.

---

# Objetivo del MVP

El objetivo principal no es predecir el comportamiento real del mercado financiero, sino ofrecer una herramienta educativa e interactiva que ayude a:

- Comprender relaciones económicas básicas
- Interpretar noticias y eventos globales
- Explorar escenarios hipotéticos
- Aprender mediante la experimentación

---

# Funcionamiento

El usuario podrá:

1. Seleccionar un evento global
2. Ajustar parámetros simples:
   - Intensidad
   - Duración
3. Ejecutar la simulación
4. Visualizar el impacto mediante gráficos e indicadores

Los resultados se representarán a través de:

- Índices bursátiles
- Commodities (petróleo, oro, etc.)
- Tipo de cambio
- Indicadores simplificados

El sistema utilizará reglas económicas generales basadas en relaciones conocidas, priorizando coherencia y comprensión antes que precisión financiera.
